
# React Nutrition with Bulma framework

## Description
Exercise with React.js showing a SPA with dynamic components and using 'Bulma' framework.

## Scripts

In the project directory, you can run:
### $ `npm install`
### $ `npm start`

- Html

- Surge.sh
